Orchida Soft UAE E-Invoicing Module

This module allows full integration between Odoo and the UAE Tax E-Invoicing system throw Orchida Soft UAE . It automates the submission, validation, and tracking of invoices directly from your Odoo environment to the E-TAX portal.

How It Works

After installing the module, configure your credentials in Settings → E-Invoice. Once set up, every validated invoice will be automatically transmitted to the ETA API.
