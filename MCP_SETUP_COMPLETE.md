# Odoo 18 MCP Setup - Complete ✅

## What's Working
Your Odoo 18 installation now has MCP (Model Context Protocol) integration with Claude Desktop.

## How to Use
1. **Open Claude Desktop** on your system
2. **Chat with Claude** and ask questions about your Odoo data
3. **Available commands:**
   - "Search for leads with high probability"
   - "Find customers in New York"
   - "Show me recent sales orders"
   - "Create a new lead for [company]"
   - "Update lead priority"

## Technical Details
- **MCP Server**: `odoo_mcp_server.py` - Connects Claude to your Odoo database
- **Wrapper Script**: `mcp_wrapper.sh` - Activates Python environment and runs server
- **Configuration**: `~/.config/Claude/claude_desktop_config.json` - Tells Claude where to find the server
- **Database**: Your Odoo 18 instance with CRM, Sales, and Partner data

## Available Tools
- `crm_search_leads` - Search and filter CRM leads
- `crm_create_lead` - Create new leads
- `crm_update_lead` - Update existing leads
- `sale_search_orders` - Search sales orders
- `partner_search` - Search customers/partners
- `partner_create` - Create new customers
